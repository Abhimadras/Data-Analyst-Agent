API Documentation

Endpoint
POST /api/
Usage Example
curl "{{ request.url_root }}api/" \
  -F "questions.txt=@questions.txt" \
  -F "data.csv=@data.csv" \
  -F "image.png=@image.png"
Supported File Types
  • Required: questions.txt (analysis tasks)
  • Optional: CSV, JSON, Parquet (data files)
  • Optional: PNG, JPG, JPEG (images)
Response Format
  • JSON arrays or objects
  • Base64 encoded plots (under 100KB)
  • Structured text answers
Features
  • Wikipedia table scraping
  • Statistical analysis & correlation
  • Data visualization generation
  • 3-minute processing time limit
